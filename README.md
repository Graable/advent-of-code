# advent-of-code

## Pushing code
To push your code to GitHub, follow these instructions:

1. Commit your code
```
git add . && git commit -am "message"
```
where message is a description of what you did

2. Push to Github
```
git push
```



## Running code
To run the code, navigate to the folder and run with the command:


1. Transpile
```
tsc -p tsconfig.json
```

2. Run
```
node day1/part1.js 
```
